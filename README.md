# ToDoApp

Simple Android ToDo App built using Kotlin and Jetpack Compose.

## Features
- Add tasks
- Delete tasks
- Task counter
- Checkbox support
- Modern UI with Jetpack Compose

## Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio
